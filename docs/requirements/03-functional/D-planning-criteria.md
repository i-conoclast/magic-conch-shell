# D. Planning — 완료 조건 (Criteria)

---

## FR-D1: 아침 브리핑 생성

- [ ] 지정된 아침 시간에 자동 실행
- [ ] 외부 태스크 시스템에서 오늘 예정 항목을 읽어옴
- [ ] 어제 미완료 항목이 포함됨
- [ ] 최근 감지된 외부 변화가 포함됨 (연동된 시스템 기준)
- [ ] 이번 주 우선순위 1~2개가 제시됨
- [ ] 사용자 응답 유도 질문 한 줄이 포함됨
- [ ] 브리핑 생성에 30초 이내
- [ ] 채팅 앱으로 전달 (Step 2 출력 채널 정의와 일치)
- [ ] 브리핑 내용이 그날 일지에도 함께 기록됨

---

## FR-D2: 대화형 플랜 확정

- [ ] 사용자가 브리핑에 응답하면 대화 흐름이 시작됨
- [ ] 3턴 이상 대화에서 이전 맥락이 유지됨
- [ ] 추가·삭제·수정 의견을 자연어로 받을 수 있음
- [ ] 각 턴마다 업데이트된 플랜 목록이 표시됨
- [ ] 명시적 "확정" 명령으로만 최종 저장됨
- [ ] 확정된 플랜은 수정 불가 (이후 수정은 새 이력으로)
- [ ] 대화 중 취소 가능

---

## FR-D3: 저녁 회고 (대화형)

- [ ] 지정된 저녁 시간 알림 또는 사용자 호출로 시작
- [ ] 오늘 확정 플랜 대비 진행 상황 확인 단계 포함
- [ ] 오늘 쌓인 엔티티 초안 목록이 회고 흐름에 통합됨
- [ ] 오늘 쌓인 제안 목록이 회고 흐름에 통합됨
- [ ] 느낀 점·배운 점 입력 가능 (선택, 강제 아님)
- [ ] 내일을 위한 한 줄 메모 옵션
- [ ] 전체 세션 5분 이내로 끝낼 수 있는 분량
- [ ] 회고 결과가 그날 일지와 장기 보존 파일에 저장됨

---

## FR-D4: 주간 리뷰

- [ ] 매주 지정 시점에 자동 실행 옵션 제공
- [ ] 한 주간 확정 플랜 대비 실행 요약
- [ ] 도메인별 기록 빈도 변화 요약
- [ ] 주목할 엔티티 활동 표시
- [ ] 다음 주 우선순위 제안 포함
- [ ] 중장기 목표와 현재 갭 체크
- [ ] 결과물이 노트 파일로 저장됨 (장기 보존)
- [ ] 사용자가 결과를 보고 후속 대화를 이어갈 수 있음
- [ ] 생성에 1분 이내

---

## FR-D5: 확정 플랜의 외부 시스템 반영

- [ ] 확정된 플랜 1건당 외부 태스크 1건 생성
- [ ] 태스크에 제목·마감일·도메인·연결 KR 포함
- [ ] 외부 시스템 실패 시 로컬 큐에 쌓임
- [ ] 자동 재시도가 정상 작동 (장애 대응 연계)
- [ ] 이미 반영된 플랜이 중복 반영되지 않음
- [ ] 반영 실패·성공 카운트가 상태 명령으로 확인 가능
