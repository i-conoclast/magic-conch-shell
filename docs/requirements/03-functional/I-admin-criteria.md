# I. Admin — 완료 조건 (Criteria)

---

## FR-I1: 상태 점검

- [ ] 단일 명령으로 전체 상태 조회 가능
- [ ] 메모리 저장소 접근 가능 여부 표시
- [ ] 인덱스·검색 엔진 상태 표시
- [ ] 외부 연동 시스템 각각의 연결 상태 표시
- [ ] 백그라운드 프로세스(watcher·스케줄러) 동작 여부 표시
- [ ] 최근 오류 로그 요약 표시
- [ ] 승인 인박스 대기 수 표시
- [ ] 로컬 큐 잔여 수 표시
- [ ] 결과가 색상·아이콘으로 상태 구분 (OK/경고/오류)
- [ ] 문제 발견 시 복구 가이드 또는 명령 안내
- [ ] 점검 수 초 이내 완료

---

## FR-I2: 인덱스 재빌드

- [ ] 사용자 명령으로 재빌드 시작
- [ ] 재빌드 진행률이 사용자에게 표시됨
- [ ] 재빌드 중에도 일부 검색이 가능 (또는 명확한 중단 안내)
- [ ] 재빌드 완료 후 검색 결과 갱신 확인 가능
- [ ] 원본 기록은 변경되지 않음
- [ ] 재빌드 실패 시 원인 안내

---

## FR-I3: 장애 시 로컬 큐 + 제한 기능 지속

- [ ] 외부 API 호출 실패 시 요청이 큐에 저장됨
- [ ] 자동 재시도가 지수적 백오프로 동작
- [ ] 재시도 횟수 상한이 설정됨 (예: 5회)
- [ ] 상한 초과 시 실패 표시 + 사용자 알림
- [ ] 큐 상태(잔여·처리중·실패·성공 이력)를 상태 명령으로 조회 가능
- [ ] 사용자가 큐 항목을 수동으로 재실행·삭제 가능
- [ ] 외부 의존 없는 기능은 장애 영향 받지 않고 정상 동작
- [ ] 외부 복구 감지 시 큐가 자동 처리 재개
