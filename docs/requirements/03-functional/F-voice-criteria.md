# F. Voice & Tone — 완료 조건 (Criteria)

---

## FR-F1: 상황별 톤 자동 판단

- [ ] 질문이 진지·가벼움·애매 중 하나로 분류됨
- [ ] 진지 도메인 키워드(커리어·면접·KR·재무·건강 등) 포함 시 컨설팅 톤
- [ ] 가벼운 키워드(점심·기분·심심 등) 포함 시 오라클 톤
- [ ] 가벼운 질문이어도 관련 기록이 있으면 하이브리드 톤
- [ ] 사용자가 톤 모드 명시 없이도 정상 작동
- [ ] 사용자가 "진지하게"·"가볍게" 등으로 톤 재조정 가능
- [ ] 판정 결과가 응답에 보이지 않게 자연스럽게 반영됨

---

## FR-F2: 컨설팅 톤 (진지)

- [ ] 응답에 선택지 2~4개 포함
- [ ] 각 선택지에 근거 1~2줄이 따라옴
- [ ] 근거에 메모리 기록 참조가 포함 (가능할 때)
- [ ] 시스템의 추천 의견이 명시됨 (하지만 강요하지 않음)
- [ ] 사용자가 결정 못할 때 후속 질문 흐름 제공
- [ ] 단일 답변 강제가 아님 (사용자가 결정 주체)

---

## FR-F3: 오라클 톤 (가벼움)

- [ ] 응답이 한 줄 혹은 2줄 이내
- [ ] 응답 사전이 파일로 관리됨
- [ ] 응답 사전에 최소 10개 이상 초기 엔트리
- [ ] 사용자가 사전을 편집해 자신의 톤에 맞게 커스터마이즈 가능
- [ ] 같은 응답이 짧은 시간에 중복되지 않도록 약간의 변주

---

## FR-F4: 동일 오라클 답 연속 방지 (streak 방지)

- [ ] 최근 3개 응답의 톤·답 이력이 저장됨
- [ ] 같은 오라클 답이 3회 연속이면 다음 응답은 컨설팅 또는 하이브리드로 강제 전환
- [ ] streak 카운터 상태가 상태 점검 명령으로 조회 가능
- [ ] 설정으로 streak 방지 비활성화 가능 (장난 이어가고 싶을 때)
- [ ] 설정 변경이 즉시 반영됨
