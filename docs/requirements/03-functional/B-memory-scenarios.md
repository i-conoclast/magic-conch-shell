# B. Memory — 사용 시나리오

---

## Scenario B1.1: 과거 맥락 소환

**Given**: 사용자가 지난달 Anthropic 관련 활동이 기억나지 않는다.
**When**: "지난 달 anthropic 관련" 질의를 입력한다.
**Then**:
- 면접 후기, 관련 메일 요약, 관련 인물 노트 등이 관련도 순으로 나열됨
- 각 결과에 발췌 + 저장 시점 + 관련 엔티티(회사·면접관) 표시
- 사용자가 결과 하나를 클릭하면 전체 기록이 열림

---

## Scenario B1.2: 의미 유사 검색

**Given**: 사용자가 "번아웃 신호"에 대해 검색한다.
**When**: 정확히 "번아웃"이란 단어가 없는 기록(예: "이번 주 계속 피곤하고 집중 안 됨")도 의미상 관련이 있다.
**Then**:
- 단어 일치하는 기록 + 의미 유사 기록 모두 결과에 포함
- 관련도 점수로 순위가 매겨짐

---

## Scenario B2.1: 도메인 필터

**Given**: 사용자가 재무 관련 기록만 찾고 싶다.
**When**: 재무 도메인 필터를 걸고 "리밸런싱" 검색한다.
**Then**:
- 재무 도메인에 속한 기록 중 관련 있는 것만 반환
- 커리어·건강 등 다른 도메인 기록은 제외됨

---

## Scenario B2.2: 엔티티 필터

**Given**: 사용자가 "Jane Smith"라는 사람 관련만 보고 싶다.
**When**: 엔티티 필터에 해당 사람을 지정하고 "follow-up" 검색한다.
**Then**:
- Jane Smith에 연결된 기록 중 "follow-up" 관련만 반환
- 다른 사람 관련 follow-up 기록은 제외

---

## Scenario B3.1: 어제 무엇 했나

**Given**: 아침에 일어나서 어제 뭐가 있었는지 확인하고 싶다.
**When**: "어제" 일지 뷰를 요청한다.
**Then**:
- 어제 캡처된 모든 기록이 시간순 나열 (오전→오후)
- 자동 생성된 어제 저녁 회고 요약도 같은 뷰에 포함
- 도메인별로 기록 개수 요약이 상단에 표시됨

---

## Scenario B4.1: 엔티티 타임라인

**Given**: 2주 후 Jane Smith와 2차 미팅이 예정돼 있다.
**When**: 사용자가 Jane Smith 엔티티 타임라인을 연다.
**Then**:
- 상단에 프로필 정보(역할·소속·첫 접촉일·마지막 접촉일)
- 시간 역순으로 관련 기록 모두 나열: 1차 면접 후기, 이메일 요약, 관련 자유 메모
- 사용자가 다음 미팅 준비에 필요한 맥락을 한 눈에 파악

---

## Scenario B4.2: 기간 필터

**Given**: 사용자가 Anthropic 관련 지난 분기 흐름을 알고 싶다.
**When**: 엔티티 타임라인에서 "지난 3개월" 필터를 건다.
**Then**:
- 해당 기간 관련 기록만 시간 역순으로 표시
- 기간 밖 기록은 숨김
- 기록 타입별 개수가 요약으로 표시됨

---

## Scenario B5.1: 오래됐지만 핵심

**Given**: 6개월 전에 쓴 자기 선언문이 현재 고민에 핵심이다.
**When**: 사용자가 관련 질의를 던진다.
**Then**:
- 관련도 점수가 높으면 6개월 전 기록이라도 결과 상단에 표시
- 시간순 정렬로 바꾸면 최신 기록이 위로 이동

---

## Scenario B5.2: 정렬 전환

**Given**: 사용자가 관련도 순으로 본 결과를 시간순으로 다시 보고 싶다.
**When**: 정렬 옵션을 "최신 순"으로 변경한다.
**Then**:
- 같은 결과 집합이 시간 역순으로 재정렬되어 표시됨
- 결과 집합 자체는 변하지 않음 (필터가 아닌 정렬만 변경)
