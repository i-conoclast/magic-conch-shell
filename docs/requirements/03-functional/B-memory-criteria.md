# B. Memory — 완료 조건 (Criteria)

---

## FR-B1: 자유 질의 검색

- [ ] 한 줄 질의 입력 후 2초 이내 결과 반환
- [ ] 결과가 관련도 순으로 정렬됨 (기본 정렬)
- [ ] 질의에 없는 유사 단어로 쓰인 기록도 결과에 포함 (의미 유사 검색)
- [ ] 한국어·영어 혼용 질의 지원 (예: "anthropic 면접")
- [ ] 각 결과에 발췌(맥락 1~2 문장)·저장 시점·도메인·엔티티 정보 표시
- [ ] 결과가 없으면 "해당 없음" 메시지와 함께 빈 목록 반환

---

## FR-B2: 도메인·엔티티 필터 검색

- [ ] 도메인 1개로 필터링 가능 (재무·커리어 등)
- [ ] 엔티티 1개로 필터링 가능 (특정 사람·회사 등)
- [ ] 도메인과 엔티티 동시 필터 가능 (교집합)
- [ ] 필터 적용 시에도 결과 포맷 일관
- [ ] 필터 결과가 0건일 때 명확한 안내 메시지

---

## FR-B3: 하루 단위 뷰

- [ ] 절대 날짜 입력으로 해당 날짜 기록 표시
- [ ] 상대 표현 지원: "오늘", "어제", "지난 금요일" 등
- [ ] 기록이 시간 순서(오전→오후)로 정렬됨
- [ ] 도메인별 그룹 뷰 옵션 제공
- [ ] 자동 생성 결과물(세션 요약·브리핑)이 같은 날짜 뷰에 통합
- [ ] 기록이 0건인 날짜에도 빈 뷰로 정상 응답

---

## FR-B4: 엔티티 타임라인 뷰

- [ ] 엔티티 지정으로 관련 기록 시간 역순 조회 가능
- [ ] 기간 필터 옵션 (지난 주·지난 달·지난 분기)
- [ ] 기록 타입별 구분 표시 (예: 면접 후기 3건, 자유 메모 12건)
- [ ] 본문에 해당 엔티티를 언급한 기록도 포함
- [ ] 엔티티 프로필 정보(역할·최근 접촉일 등)가 뷰 상단에 표시

---

## FR-B5: 관련도 정렬

- [ ] 기본 정렬이 관련도 순
- [ ] 정렬 옵션으로 시간 순(최신/오래된 순) 변경 가능
- [ ] 관련도 점수가 동률일 때 최근 기록이 상단
- [ ] 관련도 점수 산출에 의미 유사성과 단어 일치 모두 반영
- [ ] 각 결과에 관련도 점수 표시 가능 (옵션)
