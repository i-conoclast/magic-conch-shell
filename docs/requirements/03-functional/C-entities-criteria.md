# C. Entities — 완료 조건 (Criteria)

---

## FR-C1: 자동 엔티티 초안 생성

- [ ] 새 기록에서 사람·회사·직무·책 관련 고유명사가 감지되면 초안 엔티티 자동 생성
- [ ] 초안에 이름·타입·탐지 시점·첫 언급 기록 링크가 자동 기입됨
- [ ] 이미 초안 또는 정식 엔티티에 존재하는 이름은 중복 생성되지 않음
- [ ] 초안 상태로는 검색·타임라인 뷰의 공식 결과에 나타나지 않음
- [ ] 감지 정확도가 모호한 경우는 생성하지 않음 (false positive 최소화)
- [ ] 생성된 초안은 승인 대기열에 올라감

---

## FR-C2: 엔티티 승인·거절

- [ ] 저녁 회고 시간 대화에 초안 목록이 포함됨
- [ ] 각 초안마다 감지된 최소 정보와 첫 언급 발췌 표시
- [ ] 승인·거절·수정 선택지 제공
- [ ] 승인 시 정식 엔티티로 이동하고, 거절 시 삭제됨
- [ ] 승인 과정에서 추가 정보(역할·소속·관계 등) 입력 가능
- [ ] 회고에서 처리 못한 초안은 다음 회고까지 유지됨
- [ ] 초안이 쌓여 일정 수 이상이면 사용자에게 알림

---

## FR-C3: 자동 양방향 연결 (back-link)

- [ ] 기록 저장 시 지정된 엔티티와 양방향 연결 자동 생성
- [ ] 엔티티 프로필에서 연결된 기록 목록이 자동 유지됨
- [ ] 기록 본문에서 엔티티 이름 언급이 감지되면 자동 연결
- [ ] 엔티티 삭제 시 참조가 자동 정리됨
- [ ] 엔티티 병합 시 양쪽 참조가 한 프로필로 통합됨

---

## FR-C4: 엔티티 프로필 조회·수정

- [ ] 엔티티 이름·식별자로 프로필 조회 가능
- [ ] 프로필에 기본 정보, 관련 기록 타임라인, 자유 메모 영역이 표시됨
- [ ] 각 필드 편집 가능 (추가·수정·삭제)
- [ ] 자동 생성된 정보와 수동 편집한 정보가 구분되어 보존됨
- [ ] 편집 이력이 기록됨 (언제·무엇이 바뀌었는지)
- [ ] 프로필을 외부 편집기로 열어 .md 파일로 직접 편집 가능

---

## FR-C5: 동명이인·중복 처리

- [ ] 같은 이름 감지 시 자동 병합 없이 신규 초안으로 생성
- [ ] 사용자 명령으로 두 엔티티 병합 가능
- [ ] 병합 시 참조·연결·메모가 하나로 통합됨
- [ ] 병합 후 undo(되돌리기) 지원
- [ ] 분리(한 엔티티를 둘로 나누기) 명령 지원
- [ ] 중복 후보가 식별 정보로 추정되면 승인 인박스에 "중복인가요?" 제안 표시
