# H. Integration — 완료 조건 (Criteria)

---

## FR-H1: 외부 태스크 시스템 쓰기

- [ ] 확정 플랜 1건이 외부 시스템에 태스크 1건으로 생성됨
- [ ] 태스크에 제목·도메인·마감일이 필수 필드로 포함
- [ ] 관련 KR 연결 가능 (있을 때)
- [ ] 이 시스템은 태스크 **상태**를 바꾸지 않음
- [ ] 이 시스템은 외부 태스크를 **삭제**하지 않음
- [ ] 중복 반영 방지 (이미 반영된 플랜 재반영 X)
- [ ] 외부 장애 시 로컬 큐에 쌓임
- [ ] 자동 재시도 정상 작동
- [ ] 반영 성공/실패 카운트가 상태 명령으로 확인 가능

---

## FR-H2: 기존 시스템 읽기

- [ ] 캘린더에서 최근 예정 일정을 가져올 수 있음
- [ ] 외부 태스크 시스템에서 오늘/내일 예정 태스크를 가져올 수 있음
- [ ] 가져온 정보가 분 단위로 갱신됨
- [ ] 읽기 전용 — 외부 시스템 상태 변경 없음
- [ ] 연동 장애 시 이 시스템의 핵심 기능은 영향받지 않음
- [ ] 연동 상태를 상태 명령으로 확인 가능

---

## FR-H3: 채팅 앱 양방향 통신

- [ ] 시스템이 사용자에게 메시지 전송 가능
- [ ] 사용자 응답이 시스템에 수신됨
- [ ] 송수신 지연 수 초 이내
- [ ] 메시지 이력이 유지되어 대화 맥락 연속성 보장
- [ ] 채팅 장애 시 터미널 출력으로 fallback
- [ ] 채팅 재연결 시 누락된 메시지 처리

---

## FR-H4: 이메일 자동 수집 (v1.0 범위)

**MVP 범위 밖**. 해당 기준은 v1.0에서 정의·검증.

참고 예정 조건 (v1.0):
- [ ] 지정 필터로 대상 이메일 감지
- [ ] 감지된 이메일이 메모리에 자동 저장
- [ ] 관련 엔티티 자동 태깅
- [ ] 수집 기록이 브리핑·세션 맥락에 반영
