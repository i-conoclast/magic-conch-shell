# 01. Feature Catalog

**작성일**: 2026-04-19

요구사항(FR)을 실제 **사용자가 인지 가능한 기능 단위**로 묶은 목록. 기술 무관. 각 feature는 충족하는 FR들을 참조.

우선순위: **P0** (MVP 4주) / **P1** (v1.0 2~3개월) / **P2** (v2.0+)

---

## 캡처 (Capture)

### F-01. 한 줄 메모 캡처
- **설명**: 터미널·채팅·파일 타이핑 중 어느 경로로든 한 줄 메모를 즉시 저장
- **충족 FR**: FR-A1, FR-A4
- **우선순위**: P0
- **MVP 이유**: 가장 기본적인 입력 기능. 없으면 아무것도 못 함.

### F-02. 구조화 기록 (템플릿)
- **설명**: 면접·회의·실험 템플릿을 선택해 필드 기반 기록 저장
- **충족 FR**: FR-A2
- **우선순위**: P0
- **MVP 이유**: 자유 메모와 구분되는 기록이 필요 (면접 후기 등)

### F-03. 도메인·엔티티 지정 캡처
- **설명**: 캡처 시점에 도메인·엔티티 옵션으로 분류
- **충족 FR**: FR-A3
- **우선순위**: P0
- **MVP 이유**: 분류된 기록이 검색·타임라인에 활용

### F-04. 외부 편집기 파일 자동 수집
- **설명**: 지정 폴더의 마크다운 변경을 감지해 자동 수집
- **충족 FR**: FR-A4
- **우선순위**: P0
- **MVP 이유**: Step 2에서 명시한 주요 입력 수단

### F-05. 중복 감지
- **설명**: 같은 내용이 중복 저장되는 것 방지
- **충족 FR**: FR-A5
- **우선순위**: P0
- **MVP 이유**: 파일 watcher와 수동 입력 겹칠 때 필수
